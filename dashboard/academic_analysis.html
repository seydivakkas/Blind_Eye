8 Akademik İnovasyon

Blind Eye'ı literatürde öne çıkaran teknik yenilikler

1

CBAM Dikkat Mekanizması

Channel + Spatial attention ile dudak bölgesine odaklanma. Küçük modellerde büyük doğruluk artışı.

2

INT8 Post-Training Quantization

Float32 → INT8 dönüşümü ile 4× boyut küçülme, %2'den az doğruluk kaybı.

3

Witten-Bell Türkçe LM

Saf Python N-gram dil modeli. C++ KenLM bağımlılığı yok — çapraz platform.

4

Morfolojik FST Post-Processing

Türkçe son-ek kuralları ile CTC çıktısını dilbilgisel olarak düzeltme.

5

KLT-FaceMesh Hibrit Takip

6× CPU tasarrufu sağlayan forward-backward hata kontrollü hibrit landmark takibi. Pi 3 B+'da 30+ FPS.

6

Duchenne Gülümseme Tespiti

FACS Action Unit tabanlı gerçek vs. yapay gülümseme ayrımı. Orbicularis oculi kasılması analizi.

7

EAR + PERCLOS Bilişsel Yük

Göz kırpma frekansı ve kapanma süresi ile bilişsel yük / yorgunluk indeksi hesaplama.

8

WiFi TCP Gözlük Pipeline

Edge device → WiFi TCP socket → PC inference → OLED altyazı. Gerçek zamanlı, düşük latency.
