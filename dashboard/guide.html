Blind Eye Nedir?

Blind Eye, işitme engelli bireylerin günlük iletişimini desteklemek amacıyla geliştirilen, kamera görüntüsünden gerçek zamanlı Türkçe dudak okuma yapan gözlük sistemidir. TÜBİTAK 2209-A programı kapsamında geliştirilmektedir. Raspberry Pi 3 Model B+ üzerinde kamera ile konuşan kişinin yüzünü çeker, WiFi üzerinden PC'ye stream eder, PC'de derin öğrenme inference yapar ve sonucu gözlükteki SSD1306 OLED ekranda altyazı olarak gösterir.

Uçtan Uca Veri Akışı

Pi Camera v2 (IMX219)

8MP, 1080p30 — CSI 15-pin doğrudan bağlantı

WiFi TCP Stream

Pi 3 B+ → PC (640×480, 25 FPS, JPEG over TCP socket)

PC Inference (ONNX)

CNN+CBAM+BiLSTM → CTC Decoder → Türkçe kelime

SSD1306 OLED (128×64)

Altyazı HUD — WiFi üzerinden gözlüğe geri gönderilir

Tam Veri Pipeline'ı

Ham videodan eğitime kadar 7 adımlık tam pipeline

1 Test Verisi

Sentetik Video Üretimi

Sinüsoidal dudak animasyonlu test videoları.

python tools/generate_test_data.py --words 10 --videos 5 --frames 30
2 Mendeley Dataset

Gerçek Türkçe Dudak Okuma Verisi

2,335 gerçek klip — 10 kelime × ~230 klip.

doi:10.17632/4t8vs4dr4v.1 (CC BY 4.0)
3 ROI

Video → Dudak ROI Çıkarma

MediaPipe ile dudak bölgesi [96, 96, 1] float32.

python tools/preprocess_dataset.py --input data/raw --output data/processed --max-frames 30
4 Augmentasyon

6 Teknikli Veri Artırma (4×)

Time Masking, Brightness, Flip, Noise, Crop, Mixup.

python tools/augment.py --input data/processed --output data/augmented --factor 3
5 LM

Türkçe N-gram Dil Modeli

Witten-Bell smoothing ile ARPA model. C++ bağımlılığı yok.

python tools/train_lm.py --dummy --order 3
6 Eğitim

CNN + CBAM + BiLSTM + CTC Loss

AdamW + CosineAnnealing, ~2 saat (RTX 4070).

python tools/train_model.py --epochs 50 --batch-size 16 --lr 1e-3
7 Export

ONNX INT8 Quantization

PyTorch → ONNX → INT8 → 2.9 MB model.

python tools/export_to_onnx.py --quantize int8

Model Eğitimi Detayları

CNN+CBAM+BiLSTM mimarisinin eğitim süreci

Mimari Detaylar

Visual Frontend (MobileNetV3-Tiny)
  └→ 3D Conv [1, 64, T, 48, 48]
  └→ CBAM Attention (Channel + Spatial)
  └→ BiLSTM [hidden=128, layers=2]
  └→ FC → [1, T, 31] logits
  └→ CTC Decoder + N-gram LM
  └→ "merhaba" (conf: 87%)

Eğitim Hiperparametreleri

OptimizerAdamW (lr=1e-3, wd=1e-4)
SchedulerCosineAnnealing (T_max=50)
LossCTC Loss (blank=0)
Batch Size16
Epochs50
QuantizationINT8 (post-training)
Model Size2.9 MB (INT8 ONNX)

Sorun Giderme

Sık karşılaşılan hatalar ve çözümleri

ModuleNotFoundError: No module named 'cv2'
Çözüm: pip install opencv-python-headless
ONNX model dosyası bulunamadı
Çözüm: Sistem otomatik mock mod'a geçer. Gerçek model için python tools/export_to_onnx.py
WiFi stream bağlantı hatası
Çözüm: PC IP adresini doğrulayın → --server-ip 192.168.1.x, firewall 8554-8555 portlarını açın
OLED ekran görüntü vermiyor
Çözüm: I2C aktif mi kontrol edin → sudo raspi-config → Interface → I2C → Enable. Adres: i2cdetect -y 1
Pi Camera "Camera not detected"
Çözüm: sudo raspi-config → Interface → Legacy Camera → Enable, ardından reboot. CSI kablo bağlantısını kontrol edin.
